Purpose

The STM32 Image Processing Library is a C software library for STM32 MCUs providing specific functionalities that help the development of visual analysis applications; it is also a component of the STM32 Middleware level.

Here is the list of references to user documents:

  • CHM: STM32 Image Processing Library documentation

Update History

Main Changes

  • Integrated the support of YCbCr to RGB565 conversion to the STM32H7 JPEG HW decoder
  • Integrated the STM32H7 JPEG HW encoder to support file saving function
  • Removed support to YCbCr MCUs based images
  • Integrated the JPEG SW codec based on libJPEG
  • Some functions have been renamed
  • Added STM32Ipl_AllocateData() function
  • Added STM32Ipl_Copy() function
  • Added STM32Ipl_CopyData() function
  • Added STM32Ipl_Clone() function
  • Added STM32Ipl_ImageDataSize() function
  • Added STM32Ipl_DataSize() function
  • Added configuration file (stm32ipl_conf.h)
  • Fixed bugs relative to the reading of 1 bit and 4 bits BMP files
